# Configura√ß√£o para Ambiente de Staging

Este exemplo mostra como configurar um cluster Kubernetes na DigitalOcean otimizado para staging/homologa√ß√£o, servindo como ambiente intermedi√°rio entre desenvolvimento e produ√ß√£o.

## üéØ Caracter√≠sticas do Ambiente de Staging

- **Downtime**: M√≠nimo aceit√°vel, mas zero downtime prefer√≠vel
- **Recursos**: Similares √† produ√ß√£o, mas em escala menor
- **Automa√ß√£o**: Equilibrio entre controle e automa√ß√£o
- **Testes**: Ambiente para validar upgrades antes da produ√ß√£o
